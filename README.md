# 📊 C++ Array Processor & JSON Exporter

Цей проєкт — консольний застосунок на C++, призначений для виконання базових математичних та логічних операцій над масивами (векторами). Програма також генерує детальний звіт про результати роботи у форматі JSON.

## 🚀 Основний функціонал

У проєкті реалізовані власні алгоритми (без використання бібліотеки `<algorithm>`):

* **Сортування (Bubble Sort):** * `sortAscend` — сортування масиву за зростанням.
    * `sortDescend` — сортування масиву за спаданням.
* **Математичні операції:** * `sumOfEl` — підрахунок загальної суми елементів масиву.
* **Робота з унікальністю:**
    * `rmDuplicates` — створення нового масиву лише з унікальних елементів (видалення дублікатів).
    * `printDiffs` — пошук та виведення елементів, які повторюються у початковому масиві.
* **Експорт даних:**
    * `JsonOut` — ручна генерація та запис результатів (суми, різниця, масиви) у текстовий файл формату `.json`.

## 🛠 Технології

* **Мова:** C++
* **Стандартні бібліотеки:** `<iostream>`, `<vector>`, `<fstream>`, `<cstdlib>`, `<ctime>`
* **Тестування:** Google Test (GTest) для перевірки логіки методів.
* **Середовище розробки:** Visual Studio
